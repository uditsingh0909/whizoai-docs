# Webhooks API

Configure webhooks for real-time notifications.

## Create Webhook

### `POST /v1/webhooks`

Create a new webhook endpoint.

```bash
curl -X POST "https://api.whizo.ai/v1/webhooks"   -H "Authorization: Bearer YOUR_API_KEY"   -H "Content-Type: application/json"   -d '{
    "url": "https://myapp.com/webhook",
    "events": ["job.completed", "job.failed"],
    "secret": "webhook_secret_key"
  }'
```

## Webhook Events

### Available Events

- `job.completed` - Job finished successfully
- `job.failed` - Job failed
- `monitor.changed` - Monitor detected changes
- `credit.low` - Credits running low

## Webhook Payload

```json
{
  "event": "job.completed",
  "timestamp": "2025-01-15T10:30:00Z",
  "data": {
    "jobId": "job_abc123",
    "status": "completed",
    "creditsUsed": 3
  }
}
```
