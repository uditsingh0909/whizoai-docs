## Get Your API Key

1. **Sign up** at [whizo.ai](https://whizo.ai)
2. **Navigate** to your [API Keys dashboard](https://whizo.ai/app/api-keys)
3. **Create** a new API key and copy it securely

<Warning>
Keep your API key secret and never expose it in client-side code or public repositories.
</Warning>

## Make Your First Request

### Scrape a Webpage

The simplest way to get started is to scrape content from a webpage:

<CodeGroup>

```bash cURL
curl -X POST "https://api.whizo.ai/v1/scrape" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://example.com",
    "options": {
      "format": "markdown"
    }
  }'
```

```javascript JavaScript
const response = await fetch('https://api.whizo.ai/v1/scrape', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    url: 'https://example.com',
    options: {
      format: 'markdown'
    }
  })
});

const data = await response.json();
console.log(data.content);
```

```python Python
import requests

url = "https://api.whizo.ai/v1/scrape"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
data = {
    "url": "https://example.com",
    "options": {
        "format": "markdown"
    }
}

response = requests.post(url, headers=headers, json=data)
result = response.json()
print(result['content'])
```

</CodeGroup>

## Advanced Features

### JavaScript Rendering

For dynamic websites that require JavaScript:

<CodeGroup>

```bash cURL
curl -X POST "https://api.whizo.ai/v1/scrape" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://spa-example.com",
    "options": {
      "format": "markdown",
      "javascript": true,
      "waitTime": 3
    }
  }'
```

```javascript JavaScript
const response = await fetch('https://api.whizo.ai/v1/scrape', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    url: 'https://spa-example.com',
    options: {
      format: 'markdown',
      javascript: true,
      waitTime: 3
    }
  })
});
```

</CodeGroup>
