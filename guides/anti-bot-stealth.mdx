## Quick Start

### Using Stealth in API Requests

Add the `stealth` parameter to any scraping request:

```json
{
  "url": "https://protected-site.com",
  "stealth": {
    "level": "advanced",
    "enableBehavior": false
  }
}
```

### Stealth Options

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `level` | `"basic"` \| `"advanced"` \| `"maximum"` | `"basic"` | Stealth intensity level |
| `enableBehavior` | `boolean` | `false` | Enable human behavior simulation (auto-enabled for maximum) |
| `sessionId` | `string` | `auto-generated` | Reuse for consistent fingerprinting across requests |

  <Accordion title="Canvas Fingerprinting">
    Adds deterministic noise to canvas rendering to prevent tracking.

  </Accordion>

  <Accordion title="WebGL Fingerprinting">
    Spoofs GPU vendor and renderer information.

    **Platform-Aware**:
    - Windows → NVIDIA/AMD/Intel GPUs
    - Mac → Apple M1/M2, AMD GPUs
    - Linux → Mesa, NVIDIA, AMD
    - Mobile → Apple A15, Qualcomm, ARM GPUs
  </Accordion>

  <Accordion title="Audio Context">
    Injects noise into audio fingerprinting APIs.

    Prevents tracking via:
    - AudioContext frequency data
    - Oscillator patterns

  </Accordion>

    - 1920x1080, 1366x768, 1440x900, etc.

## Troubleshooting

### Still Getting Detected?

<Steps>
  <Step title="Verify Stealth Level">
    Make sure you're using `advanced` or `maximum` for protected sites.
  </Step>

  <Step title="Enable Behavior Simulation">
    Add `"enableBehavior": true` for heavily protected sites.
  </Step>

  <Step title="Use Proxies">
    Enable proxy rotation to avoid IP-based blocking.
  </Step>

  <Step title="Add Wait Times">
    Increase `waitTime` to 3-5 seconds for complex pages.
  </Step>

  <Step title="Maintain Session Consistency">
    Use the same `sessionId` for multiple requests to the same domain.
  </Step>
</Steps>

### Detection Indicators

If you're still being detected, check for:

- **Network-level blocking**: IP reputation, rate limiting
- **TLS fingerprinting**: Use rotating proxies
- **Behavioral analysis**: Enable `enableBehavior: true`
- **JavaScript challenges**: Increase `waitTime`
