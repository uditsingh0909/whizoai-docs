### 3. API Migration

**Update Your Integration Code**:

<Tabs>
  <Tab title="Basic Scraping Migration">
    **Step 1**: Update your base URL and authentication
    ```javascript
    // Before
    const OLD_API = 'https://old-service.com/api';
    const headers = { 'X-API-Key': 'old-key' };

    // After
    const NEW_API = 'https://api.whizo.ai/v1';
    const headers = { 'Authorization': 'Bearer YOUR_WHIZO_KEY' };
    ```

    **Step 2**: Update request format
    ```javascript
    // Before (varies by provider)
    const oldRequest = {
      url: 'https://example.com',
      render_js: true,
      format: 'text'
    };

    // After (standardized)
    const newRequest = {
      url: 'https://example.com',
      options: {
        javascript: true,
        format: 'markdown'  // Better structured output
      }
    };
    ```

    **Step 3**: Update response handling
    ```javascript
    // Before (provider-specific format)
    const content = response.data.content;

    // After (standardized format)
    const { content, metadata } = response.data;
    // metadata includes title, description, statusCode, etc.
    ```
  </Tab>

  <Tab title="Advanced Features Migration">
    **Batch Processing**:
    ```javascript
    // Before: Multiple individual requests
    const results = await Promise.all(
      urls.map(url => scrapeOldService(url))
    );

    // After: Efficient batch processing
    const response = await fetch('https://api.whizo.ai/v1/batch', {
      method: 'POST',
      headers: { 'Authorization': 'Bearer YOUR_KEY' },
      body: JSON.stringify({
        urls: urls,
        options: { format: 'structured' }
      })
    });
    ```

  <Tab title="Error Handling Migration">
    **Improved Error Handling**:
    ```javascript
    // Before: Inconsistent error formats
    try {
      const result = await oldService.scrape(url);
    } catch (error) {
      // Varies by provider
      console.log('Error:', error.message);
    }

    // After: Standardized error format
    try {
      const response = await fetch('https://api.whizo.ai/v1/scrape', {
        method: 'POST',
        headers: { 'Authorization': 'Bearer YOUR_KEY' },
        body: JSON.stringify({ url })
      });

      if (!response.ok) {
        const error = await response.json();
        throw new Error(`${error.error}: ${error.details}`);
      }
    } catch (error) {
      // Structured error handling
      if (error.status === 429) {
        console.log('Rate limited, retrying...');
      } else if (error.status === 402) {
        console.log('Insufficient credits');
      }
    }
    ```
